# Node Learning

This repository contains basic JavaScript practice files for learning Node.js.
The examples focus on core JavaScript concepts such as arrays, objects, object
methods, and common data operations.

## Requirements

- Node.js installed on your system
- A code editor such as Visual Studio Code

## Files

| File | Description |
| --- | --- |
| `day1.js` | Practice with arrays and array methods |
| `day3.js` | Practice with objects and object methods |

## Topics Covered

### `day1.js`

- Creating arrays
- Adding and removing array items
- Looping through arrays
- Finding item indexes
- Slicing and splicing arrays
- Sorting arrays
- Finding the maximum number
- Using `reduce()`
- Using `map()`

### `day3.js`

- Creating objects
- Accessing object properties
- Updating object values
- Adding and deleting properties
- Working with nested objects
- Using `Object.keys()`
- Using `Object.values()`
- Using `Object.entries()`
- Copying objects with `Object.assign()`
- Checking object properties
- Freezing objects with `Object.freeze()`

## How to Run

Open a terminal in this folder and run:

```bash
node day1.js
```

To run the object practice file:

```bash
node day3.js
```

## Purpose

The purpose of this repository is to practice JavaScript fundamentals step by
step while learning how to run JavaScript files using Node.js.
